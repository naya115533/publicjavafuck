자바 API(Application Programming Interface)
주요한 기능들을 미리 구현한 클래스 라이브러리의 집합

자바 패키지
서로 관련된 클래스들을 분류하여 묶어 놓은 것

ctrl + f11 실행
ctrl + =/- 텍스트 확대/축소
ctrl + / 주석 설정/해재

자바의 특성
1)플랫폼 독립성
자바 프로그램은 플랫폼에 상관없이 어디서든지 실행

2)객체지향
상속성,다형성,캡슈화

3)클래스로 캡슐화
4)소스와 클래스 파일
5)하나의 클래스 안에 여러개의 클래스 생성 가능 (저장은 상위 클래스 이름으로)

식별자
클래스,변수,상수,메소드 등에 붙이는 이름
특수문자x 언더바 가능

자바의 데이터 타입
boolean 부울1비트 (참 or거짓)
빼고 나머지 c언어랑 같음

변수
자료를 저장하기 위한 공간

15           (10진수)
015         (8진수)
0x15       (16진수)
0x0101  (2진수)
